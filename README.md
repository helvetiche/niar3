# NIATools

A comprehensive workspace platform designed to streamline document processing and administrative workflows. NIATools provides a suite of integrated tools for managing billing units, processing documents, and handling file operations efficiently.

## Overview

NIATools is a web-based application that centralizes common document processing tasks into a single, unified workspace. The platform is built with security and user management at its core, ensuring that sensitive operations are protected through role-based access control.

## Key Features

### Workspace Hub
Central dashboard providing quick access to all available tools and features. Navigate seamlessly between different functionalities without losing context.

### IFR Scanner
Automated document scanning and data extraction tool. Upload IFR documents and let the system process and extract relevant information automatically. Supports batch processing for multiple files.

### Consolidate Billing Unit
Merge and consolidate multiple billing unit documents into a single, organized file. Customize folder structures and apply templates for consistent output formatting.

### Merge Files
Combine multiple documents into unified files:
- Merge PDF files with custom page ordering
- Combine Excel workbooks while preserving sheet structures
- Rename and organize sheets in merged Excel files

### LIPA Summary
Generate comprehensive summary reports from LIPA documents. Scan multiple files and produce consolidated reports with relevant data points.

### Template Manager
Create and manage reusable templates for document processing operations. Templates can be shared across different tools to maintain consistency in output formatting.

### Account Management
Administrative interface for managing user accounts and permissions. Control access levels and monitor user activities within the system.

## User Roles

The platform supports three user roles with different permission levels:

- **Super Admin**: Full system access including user management and all tools
- **Admin**: Access to all document processing tools and template management
- **User**: Access to core document processing features

## Security

All operations are protected through authentication and authorization checks. User sessions are managed securely, and sensitive operations require appropriate permissions. The platform implements rate limiting to prevent abuse and ensure fair resource allocation.

## Getting Started

1. Access the platform through your web browser
2. Log in with your credentials
3. Navigate to the Workspace Hub to view available tools
4. Select the tool that matches your current task
5. Follow the on-screen instructions for each tool

## Document Processing Workflow

Most tools follow a similar workflow:

1. Upload your source files
2. Configure processing options or select templates
3. Review and adjust settings as needed
4. Execute the operation
5. Download the processed results
